# Creator Studio Design Guidelines

## UI/UX Principles
- **KISS & YAGNI**: 在进行任何 UI/UX 重构或设计更新时，必须严格遵守 KISS (Keep It Simple, Stupid) 和 YAGNI (You Aren't Gonna Need It) 原则。
- **避免过度光效**: 优先保证直观明了和紧凑排版，绝对杜绝不必要的巨型阴影、光晕（Glow/Pulse/Blur外发光）和侵入式全屏遮盖，保持“专业生产力工具”的克制与冷峻感。
- **直观层级**: 永远确保内容的核心可见性。不要让提示或背景特效喧宾夺主。
