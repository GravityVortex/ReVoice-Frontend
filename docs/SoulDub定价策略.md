# SoulDub定价策略

# 原则

- 登录前允许上传视频
- 登录后才可生成翻译视频
- 未登录用户在点击生成时，引导用户到注册/登录页面
- 新注册用户可赠送积分
- 视频翻译每分钟消耗3积分
- 订阅策略要简单明了，不要让用户有选择困难
- 订阅分为免费版、标准版、高级版、企业版(没啥用，但是个噱头，看上去专业可信)
- 包月和包年用户权益一致，包年用户可免两个月订阅费
- 单独买积分要比包月和包年略贵，否则用户宁愿单独买积分，而不是包月和包年了

# 定价策略

- 包月：$39/月，即$468/年
- 包年：$390/年，即享受16.5%折扣，一年可节省78块钱，相当于免费送2个月

# 积分策略

- 包月用户

$39/月，300积分，即$0.13/积分，即可用100分钟

- 包年用户(享受15%的折扣)：

$33/月，300积分，即$0.11/积分，即可用100分钟

- 单独买积分（可累积，次月不清零）

积分单价：$0.2/积分

提供三档选择：60积分($12)、200积分($38,5%折扣)、500积分($90,10%折扣)
